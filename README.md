# Firebase_ReadData
建立目的 : 引用 google 官方技術文件示範如何以 Android 操作 FireBase (Real Time Database)。 

# Firebase - Real Time Database 介紹
google 建立的雲端資料庫服務，以 JSON 格式儲存資料為 NoSQL 非關聯式資料庫，Android 可以透過 HTTP requests 存取 Real Time Database 資料庫內容。
